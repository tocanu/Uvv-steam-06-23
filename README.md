# Atividade UML-GUI-Jogos

Projeto da disciplina de Programação Orientada a Objetos II

## ?? Descrição

Este repositório contém a implementação de dois jogos em C# Windows Forms:

- **Jogo 06**: Jogo de luta estilo Punch-Out
- **Jogo 23**: Jogo Snake clássico

## ?? Objetivos

- Aplicar conceitos de Programação Orientada a Objetos
- Criar diagramas UML (casos de uso e classes)
- Desenvolver interface gráfica com Windows Forms
- Implementar melhorias e novas funcionalidades

## ??? Tecnologias

- C# 14.0
- .NET 10
- Windows Forms

## ?? Estrutura do Projeto

```
AtividadeUMLJogos/
??? Jogo06/          # Projeto do Jogo 06
??? Jogo23/          # Projeto do Jogo 23
??? README.md
```

## ?? Como Executar

1. Abra a solução `AtividadeUMLJogos.sln` no Visual Studio
2. Selecione o projeto desejado (Jogo06 ou Jogo23)
3. Pressione F5 para executar
